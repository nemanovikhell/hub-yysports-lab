# hub-yysports-lab

这是一个用于归档和发布多个独立 HTML 页面的仓库。页面内容涵盖运动、休闲、数据展示等方向，适合作为演示、参考或轻量级工具页使用。

## 项目简介

本仓库旨在集中存放和展示一批独立开发的 HTML 页面，每个页面均为自包含的静态文件。项目不绑定特定网站或服务，仅作为通用页面归档与发布平台。

## 目录说明

```
.
├── pages/          # 存放所有 HTML 页面文件
├── assets/         # 公共资源（图片、样式、脚本等）
└── README.md       # 本文件
```

- `pages/` 目录下每个子文件夹代表一个独立页面，包含该页面的完整 HTML 及所需资源。
- `assets/` 用于存放跨页面共享的公共文件，如图标、基础样式或通用脚本。

## 页面归档说明

- 每个页面均为独立可运行的 HTML 文件，不依赖外部服务或数据库。
- 页面内容可能涉及运动数据展示、休闲工具、信息卡片等，具体功能见各页面内说明。
- 页面文件命名采用英文小写及连字符，便于直接通过链接访问。

## 维护说明

- 欢迎提交 Issue 反馈问题或建议。
- 如需新增页面，请在 `pages/` 下创建新文件夹并放置 `index.html`，同时可附带必要资源。
- 公共资源请优先放置在 `assets/` 中，避免重复。
- 请勿在页面中包含任何商业推广或诱导性内容。

## 使用方式

你可以直接通过 GitHub Pages 或本地服务器浏览 `pages/` 下的页面。将仓库克隆到本地后，使用任意静态服务器工具（如 `python -m http.server`）即可查看。
